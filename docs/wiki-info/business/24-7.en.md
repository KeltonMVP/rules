## Characteristics

- The B-Zone RPG server has a total of 16 24/7 stores spread across the map.
- Each 24/7 store is marked on the map with a red letter "S".

## Products

The following list contains all items available on the shelves of the B-Zone RPG server's 24/7 stores:

- Mobile phone - $1500.
- MP3 Player - $2000.
- Dice - $500.
- Fuel canister - $2000.
- Camera - $100.
- Cigarettes - $100.
- Lighter - $300.
- Radio station - $5000.
- Fireworks - $1000.
- Smoke grenades - $200.
- Fishing rod - $400.
- Bait - $20.

*Legend: Product name - Product price*

## Specific Commands

The business has two commands:

### /buy

By using this command, the server will display to the one typing the command a dialog with items, with the help of which the items mentioned above (the "Products" category) can be purchased.

The command works anywhere inside the business.

*See dialog example in image #1.*

### /lotto

By using this command, the server will display the lottery system interface, more details are available at Lotto.
