# Vehicule Cash

- Vehiculele standard (cash) ale serverelor RPG sunt vehiculele ce pot fi cumparate folosind moneda jocului ($) incepand cu nivel 3.
- Vehiculele standard (cash) pot fi vandute altor jucatori prin utilizarea comenzii /trade. Acestea pot fi vandute de asemenea la Dealership pentru 60% din valoarea acestora.
- Lista intreaga a tuturor vehiculelor standard (cash) prezente in Dealership:

## Legenda

- Pret Dealership - Pretul pentru care poti cumpara un vehicul nou din Dealership.
- Returnare Dealership - Suma de bani pe care o primesti in cazul in care iti vinzi masina Dealership-ului (/v -> vehicul -> '''Vinde''').
- Returnare Dealership Gold - Suma de bani pe care o primesti in cazul in care iti vinzi masina Dealership-ului (/v -> vehicul -> '''Vinde'''), iar aceasta este cumparata cu Gold.
- Viteza maxima - Viteza maxima pe care vehiculul o poate atinge.
- Provenit din - Nume de masini reale asociate prin asemanare cu cele din San Andreas.
- Numar de locuri - Numarul maxim de jucatori ce pot intra in acelasi timp in vehicul (cu tot cu sofer).
- Pret in gold - Daca apare in lista informatiilor unui vehicul, semnifica faptul ca vehiculul respectiv este achizitionabil folosind moneda reala, gold-ul (deci poate fi gasit si pe lista vehiculelor gold) si afiseaza cantitatea de gold necesara achizitionarii.
- Tunabil - In cazul in care vehiculul este tunabil, este afisat numele tuning-ului de care apartine, iar in caz contrar este afisat mesajul "Nu este tunabil".
