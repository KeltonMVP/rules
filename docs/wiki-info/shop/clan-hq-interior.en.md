## General Information

!!! warning "Warning"
    The gold prices presented on this page are standard. It is possible that in-game products may be cheaper if you make the purchase during a discount period.

The **Clan HQ Interior** option from /shop allows changing the current clan interior with a new predefined interior.

The standard price for changing the clan HQ interior is **600 Gold**.

To be able to modify the HQ interior, you must be the owner of a clan of **at least level 7**.

At clan level 7, you will automatically receive 1 Free Clan HQ Interior with which you can modify the interior. Other Clan HQ Interiors will need to be purchased from /shop.

Depending on the clan level, you can only select certain available interiors.

If you want images of certain interiors, you can access [this link](http://weedarr.wikidot.com/interior).

Using the **/claninfo** command you will be able to see how many Clan HQ Interiors you have available.

## Usage

- The clan owner can modify the HQ interior starting from clan level 7, positioning themselves in front of the HQ and using the **/interiorhq** command.
- Using the above command will open a dialog to select the new HQ, where you will see a list of predefined interiors.
- The number of interiors increases as the clan level is higher and unlocks new interiors.
- The owner then selects one of the desired interiors and will receive a confirmation dialog.
- Confirming through the **Yes** button will automatically modify the HQ interior.

## How to Buy?

Follow these steps:

1. Access the community shop using the **/shop** command.
2. Select from the displayed list an item whose name follows this model: **Clan HQ Interior - price Gold**.
3. You will then receive a confirmation dialog that you want to purchase this product.
4. If you confirm and click the **Order** button, you will receive 1 Clan HQ Interior.
5. **Congratulations!** You now have the possibility to change your clan HQ interior.
