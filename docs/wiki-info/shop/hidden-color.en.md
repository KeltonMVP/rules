# Hidden Color

<figure markdown="span">
    ![Hidden Color Menu](../../images/wiki/general/Hidden_menu.jpg){ width=400" }
</figure>

If you want a shiny vehicle with a special color, different from everyone else's, then you need to buy a set of hidden colors for your vehicle.

!!! note "Note"
    Learn how you can buy gold by following the steps on [this page](buy-gold.md).

!!! warning "Warning"
    The gold prices presented on this page are standard. It is possible that in-game products may be cheaper if you make the purchase during a discount period.

## Price and Duration

- The price of a set of two colors (primary and secondary) costs 600 gold (see /shop).
- Once the two colors are purchased, they will remain usable for your vehicle forever, but if you want to sell the vehicle to another player, the color set will disappear.

## How to Buy?

In just a few simple steps, your vehicle can have such a color set:

1. Access the community shop using the **/shop** command.
2. Select the following item from the displayed list: **Hidden color - 600 Gold**.
3. The server will display some information about what you need to do to get the color set.
4. After reading the information, press the **Order** button.
5. If you have an active red checkpoint on the map, the server will abandon your order, requiring you to use the **/cancel checkpoint** command to cancel it. Once the checkpoint is canceled, follow the steps from the beginning.
6. The server will set a red checkpoint on the map, you need to get there with the vehicle you want to install the color set on.
7. If you canceled the red checkpoint provided by the **Order** button, don't worry. Type the /hiddencolor command and the checkpoint will appear again.
8. To enter the hidden color selection menu, you must type the **/hiddencolor** command while being near the CarColor business.
9. After finishing the selection of the two hidden colors, press the **Buy Color** button.
10. **Congratulations!** You have a nicer vehicle.
