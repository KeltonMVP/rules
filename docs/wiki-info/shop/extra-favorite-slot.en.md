# Extra Favorite Slot

An Extra Favorite Vehicle Slot (Extra Favorite Slot) represents the player's ability to purchase one more slot for the favorite vehicles list.

Such a slot can be added to your garage for the amount of **300 Gold** per slot.

## How to Buy It?

Follow these steps:

1. Access the community shop using the **/shop** command.
2. Select from the displayed list an item whose name follows this model: **Extra Favorite Slot - price Gold**.
3. You will then receive a confirmation dialog that you want to purchase this product.
4. If you confirm and click the **Order** button, you will receive 1 Extra Slot in the favorite vehicles list.
5. **Congratulations!** You can now add another vehicle to the list.
