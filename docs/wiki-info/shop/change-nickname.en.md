Do you need a new identity? The community shop is always at your disposal (/shop).

!!! note "Note"
    Learn how you can buy gold by following the steps on [this page](buy-gold.md).

!!! warning "Warning"
    The gold prices presented on this page are standard. It is possible that in-game products may be cheaper if you make the purchase during a discount period.

## Price

- The price of changing the name is **600 Gold**.

## How to Buy?

Follow these steps:

1. Access the community shop using the **/shop** command.
2. Select the following item from the displayed list: **Change nickname - 600 Gold**.
3. The server will display some information about the name change process.
4. After reading the information, press the **Order** button.
5. Immediately after pressing the **Order** button, the server will display a box where you need to type the new name and press the **Ok** button.
6. **Congratulations!** You now have a new identity in-game.

!!! warning "Remember"
    - **On the server's website, the history will remain undeleted, the old name remaining in the log.**
    - **Using vulgar names is punishable by admins with a permanent ban.**
