# Clear Faction Punish

If you need to join a certain faction with your friends, but the punishment from the last faction (Faction Punish) doesn't allow it, then quickly access the B-Zone community shop and get rid of this unpleasant incident.

!!! note "Note"
    Learn how you can buy gold by following the steps on [this page](buy-gold.md).

!!! warning "Warning"
    The gold prices presented on this page are standard. It is possible that in-game products may be cheaper if you make the purchase during a discount period.

## Price

- The price of clearing 10 FP (Faction Punish) points costs 200 gold.
- If you have less than 10 FP points, you will still be charged 200 gold.

!!! info "Note"
    If for example you buy Clear 10 FP when you have 0 FP, it will be set to -10, so the money is not lost.

## How to Buy?

Follow these steps:

1. Access the community shop using the **/shop** command.
2. Select the following item from the displayed list: **Clear 10 FP - 200 Gold**.
3. The server will display some information about the purchase you are about to make.
4. After reading the information, if everything is in order, press the **Order** button.
5. **Congratulations!** The purchase was successful.
