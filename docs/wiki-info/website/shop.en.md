# Shop

This page displays all products that you can purchase with Gold in game.

## Page structure

### Page title

<figure markdown="span">
  ![Shop Products](/images/wiki/website/shopproducts.png){ width="300" }
  <figcaption>Shop Products</figcaption>
</figure>

Under the page title we will be informed if there are active discounts on the servers as well as a counter that indicates how long these discounts are still active.

### Page content

Below we will find each product available in the game categorized by their type and structured as follows:

* **Product Name:** displays the product name in game.
* **Price:** displays the product price. (when there are active discounts on the shop they will be displayed)
* **Info:** displays the main information about what the respective product offers us.

**Premium Account:** we will find all available packages that offer us a Premium Account.

**Cash Money:** we will find all available packages that offer us cash money directly in game.

**Vehicle Products:** we will find all vehicles that we can purchase from the Premium Dealership as well as all products related to vehicles.

**House Products:** we will find all products with interiors and garages for houses.

**Clan Products:** we will find all products for creating a clan as well as those related to the clan.

**Skin Products:** we will find the three types of skins that can be purchased as well as the two types of skin tickets.

**Account Products:** we will find all products with which we can perform different actions on our account or which we can use in different actions.
