!!! info "Note"
    The latest updates are marked with the red flag, **New**.

* The B-Zone community emphasizes organization so that players can find what they want in just a few clicks. For this reason, the RPG servers' websites have the **Updates** page where every change made to the website or server is noted.
* The main page is for RPG server updates. It also includes 2 buttons to display updates made to the website or updates made to the server.

* Every time the server or website is updated, players will receive a notification in their own accounts.
* Updates are structured in such a way that visitors can quickly find a certain update made at a certain time.
* Their structure consists of introducing information into well-optimized spoilers having a title according to the following model: **Update date - Update version**.
