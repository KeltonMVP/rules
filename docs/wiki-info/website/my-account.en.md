On the left side of the site there is a column containing several tabs with which players can find information strictly about their own accounts.

## Messages

On this page you can view the private messages you have received from senior staff, respectively those you have sent to players for various reasons. Only level 4+ admins can send you messages on the website. Messages will be sent when direct communication with the respective player involved in a certain situation is necessary (clan leaders, hacked accounts, etc.) and is not available in-game at the desired time.

The page displays all messages you have received/sent, respectively in which you have been added, along with the following information:

* **Time:** represents the date when the message was sent.
* **From / To:** represents from whom to whom the message was sent.
* **Participants:** represents the total number of participants in the message.
* **Subject:** represents the title of the message.
* **Message Preview:** represents a small part from the beginning of the message content.
* **Last Reply:** represents the date when the last reply was given in the message.
* **View:** represents a button through which you can see the entire content of the message.

The page with the message content also displays the following information and options:

* In the upper left the title of the message.
* In the upper right displays a representative image of the skin of each participant in the message.
* **For STAFF, under the images mentioned above, the following options are also displayed:**
    * Option to add other participants in the message using their exact names.
    * Option to remove someone added from the message (in case you made a mistake with the name).
    * Option to close/open the respective message.
* Following which the content of the message will be displayed in the page in the following order:
    * The first message in the page that is displayed represents the content that was initially written in the message when it was sent.
    * The following messages in the page represent the answers of each participant in descending order, from the most recent to the oldest.
* Each sent reply contains the option to delete that reply, which will remain visible only to STAFF along with the date when it was deleted and by whom.
* Once a message has been closed by STAFF, no further replies can be added by anyone until it is reopened.
* Sent/received messages **cannot be deleted**. They remain in history as evidence for the purpose of their sending.
    * Occasionally there is the possibility for very old messages to be deleted permanently by the server management when it is considered that they are no longer needed.

## Marathon

On this page we can see all the information about the marathon when there is one in progress as follows:

* Marathon season number.
* Current version we have at the marathon.
    * Players who unlock the PRO version benefit from rewards from both versions, regardless of when they decide to unlock the PRO version.
    * Unlocking the PRO version can only be done in-game using the **/marathon** command > Get Pro and paying the amount of 3,000 Gold.
* The day we are at with the marathon along with the number of uncollected prizes from the marathon.

Under this information, each day of the marathon, the prizes from both versions as well as the current number of points out of the required number of points for that day are found.

Under the marathon prizes, the badge you receive automatically on your website profile when completing the marathon completely can be found.

At the bottom of the page we can also find a table with all the actions we can do on the server to get points as follows:

* **Activity:** represents the actual action we can do.
* **Points:** represents the number of points we receive when we do that action.
* **Maximum Limit:** represents the maximum number of points we can accumulate per day from that action.
    * Actions that do not have an imposed limit mean they can be done until you reach the current day with the marathon.
    * The limit reached by actions is automatically reset at midnight (00:00).

## Achievements

Achievements are goals you can accomplish by completing tasks. You will be rewarded for completing them.

There are two types of achievements, for a total of **280** achievements:

* Those that can be obtained only once and others that measure how many times a certain action has occurred.
* For example, an achievement that is awarded once is to reach level 20, and an achievement that is counted is to organize an event.

## Faction

The factions page contains various information and logs regarding the factions in which the player is active or has been active in the past.

There is a special table on this page that contains main data about the faction the player is part of:

* The faction the player is part of
* The date when the player joined
* The player's rank in that faction
* Number of warnings received in the faction
* Number of penalties in the faction (Faction Punish).

### Top 5 faction ranks

In this table, the last 5 highest ranks obtained in the faction will be displayed.

### Latest faction applications

The table displays the latest applications the player has submitted:

* Date when the application was submitted
* Faction name
* Application status (Accepted/Failed tests/Rejected/Invited to faction)
* A button to view the application

### Latest Faction logs

The table in this category contains information collected when a faction leader executes faction-specific commands (example: Invitation to faction):

## Faction activity

On this page, any member of a faction can view personal statistics accomplished in the faction during that week.

There are two branches into which weekly statistics can fall:

1. Total items registered as valid in the statistics.
2. Total items registered as invalid in the statistics.

**Note:** If the faction leader considers that a certain item of the statistics is corrupt/fraudulent/or has other problems, they can move it to the list of invalid items. Each faction has a certain policy regarding the validity of statistics items.

## Faction reports

Any player can read their own reports accomplished over time in factions, with structured data displayed in a table informing you about:

* Name of the faction in which the respective activity report was registered
* Time interval (one week)
* Number of points accumulated for the report

### Blacklist

The first table on the page will display the factions that have added you to the blacklist and the information needed to remove you from that list. The next table will display all factions to which you have been added to the blacklist over time.

## Security

Being one of the most important pages of any account, here each player can maintain their own security by changing their email address or activating their two-step authentication. By accessing the page, pressing on the first option you have the possibility to view the current email address associated with your account, as well as access the **Change** button to change your email address.

If you want to change your email address, but the current email address is not confirmed, then to do this you will use the /changeemail command directly in-game. For more details about changing the email address associated with the account, in-game, access the [Email](../other-systems/email.md) page. If the current email address is confirmed, then the website allows changing it using the **Change** button.

If you select the second option, you will receive an email with the account pin if you have a valid email on the account.

If you select the third option, **Activate Two Factor Authentication**, and press the **Activate** or **Deactivate** button (depending on the case) you will be able to activate/deactivate two-step authentication. More information about this as well as an installation tutorial can be found [here](https://forum.b-zone.ro/topic/438040-2fa-autentificare-in-doi-pasi/).

## Badges

On this page you can manage the badges you have unlocked. In the first part of the page you find some instructions on how to use the system and at the bottom you find a table with badges along with the following information:

* **Badge Icon:** displays the image of the respective badge.
* **Badge Name:** displays the name of the respective badge.

* **Sort:** displays a box with the order number of the respective badge (the order being **descending**).
    * For a badge to be displayed first (from top to bottom) you must assign it the value **6**.
    * For a badge to be displayed second (from top to bottom) you must assign it the value **5**.
    * For a badge to be displayed third (from top to bottom) you must assign it the value **4**.
    * For a badge to be displayed fourth (from top to bottom) you must assign it the value **3**.
    * For a badge to be displayed fifth (from top to bottom) you must assign it the value **2**.
    * For a badge to be displayed sixth (from top to bottom) you must assign it the value **1**.
    * If you don't want a specific place for a certain badge you must assign it the value **0**.
* **Hidden:** displays a box that if you check, that badge will **not** be displayed on the profile anymore.

After you have made the desired modifications you need to use the **Save** button to save.

**Note:** A maximum of 6 badges can be displayed on the profile.
