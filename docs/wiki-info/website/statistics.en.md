# Statistics

## General characteristics

The main page of statistics is represented by the top players. The top is updated based on the level of all players, being displayed in a paginated table.
Also in sub-pages you can find various statistics about the server such as businesses & houses on the server, list of bans given, etc.

[**Statistics** RPG](http://www.rpg.b-zone.ro/stats)

## Top Achievements

In this category, each type of achievement is presented along with their number.

* General Achievements (30)
* General Faction Achievements (7)

* Gangs Achievements (13)

In this category there is a paginated table that includes columns arranged with players who have played the most hours over a period of one month.

The table contains the following information:

* Position occupied.
* Player name.
* Hours played in the previous month.
* Player level.
* Faction they belong to.
* Rank they have in it.

The table contains the top 15 most active players, to see the continuation of the list you must go to the following pages.

## Weekly referrals

In this category there is a paginated table that includes columns arranged according to the number of hours spent in game summed by all recruits of a player over a period of one week. Also, above the table the period in which the top is included is specified.

The table includes:

* Player's position in top
* Name of the player who has recruits
* Total number of hours played by recruits
* Total number of recruits of the player

## Top referrals

Here total data regarding players' recruits are displayed. The top is calculated based on the total number of experience points that a player has collected from their recruits leveling up.

The data is structured in a table using the same method as in the case of the weekly recruits top.

## Top Gold Rewards

In this category there is a paginated table that includes columns arranged with the **first 50 players** who have the most hours in the current month.

The following information is displayed in the table:

* Player name
* Position occupied
* Level
* Respect points
* Their faction
* Hours played in the current month

## Houses

This sub-page displays all available interiors on the server.
Each interior contains a representative image to see what it looks like.
Each interior contains the following information:

* Interior type.
* Interior ID.
* How many houses exist on the server with that interior.

If you press on the number of houses with that interior existing on the server, you will be redirected to another page with the following information:

* Player name.
* Hours played in the current month.
* Player level.
* Interior description.
* House level.
* State price of the house.
* Whether or not it is for sale.
    * If it is for sale, the price set by the owner in game appears.
* Date of their last login to the server.
* Location on map.

If you press the **Map** button, you will be redirected to a page with the in-game map and a red dot with its exact location.

## Vehicles

This sub-page also includes other sub-categories to divide the totality of vehicles into:

* Cars
* Motorcycles
* Boats
* Aircraft

The four sub-categories include all vehicles available in the DealerShip, both premium and non-premium.

Each vehicle has some information about it under its own image:

* Model name
* Price in DealerShip
* Number of vehicles of this type owned by all server players.

If you press on the number of vehicles of a certain type owned by server players, the website will redirect you to a page containing a table with information.

The information is structured as follows:

* Name of the owner of that type of vehicle
* Owner's level
* How the vehicle was purchased (real currency store or dealership)
* Vehicle price
* Amount of money the player receives if selling the vehicle to DealerShip.
* Last date the player logged into the server.
