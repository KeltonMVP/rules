# Locations

The B-Zone RPG server has a single command that includes absolutely all important locations, as well as faction headquarters.

## /locations | /gps

Using this command, the server will display a frame that includes 10 elements:

- Important locations
- Faction HQs
- Houses
- Businesses
- Clan HQs
- ATMs
- 24/7s
- Gas stations
- Clothing stores
- Food stores

<img src="https://i.postimg.cc/TKmjFwTp/5P2YTt-J.png" style="width: 300px;background: rgb(247, 247, 247);border: 0.5px solid #BDBDBD;padding: 0px, 0px, 10px, 0px;">

### Important locations

If you select the first element from the list, you will be shown a list of all important server locations.

_Example: Banks, CNNs, Gun shops, DMV, City Hall, etc._

<img src="https://i.postimg.cc/v1bnKtPt/Locations2.png" width="300px">

_Important locations_

### Faction HQs

If you select the second element from the list, you will be shown a list of all server faction headquarters.
_Example: 69 Pier Mobs HQ, Hitmen Agency HQ, Tow Truck Company HQ, National Guard HQ._

<img src="https://i.postimg.cc/wRyRNfvv/Locations3.png" width="300px">

_Faction Headquarters_

### Houses

If you select the third element from the list, you will be shown a dialog in which you must enter the ID of the house you want to locate.

<img src="https://i.postimg.cc/9Q8KhNvy/hound-find.png" width="300px">

### Businesses

If you select the fourth element from the list, you will be shown a dialog in which you must enter the ID of the business you want to locate.

<img src="https://i.postimg.cc/zXLR7Lq5/business-find.png" width="300px">

### Clan HQs

If you select the fifth element from the list, you will be shown a dialog in which you must select the clan you want to locate.

_Note: For more information about clan HQ IDs, use the [/clanhqs] command._

<img src="https://i.postimg.cc/c4z4jLX0/gta-sa-Kl-YO7M3XFs.png" width="400px">

### ATMs

If you select the sixth element from the list, you will be shown a list of all server ATMs.

<img src="https://i.postimg.cc/JhXf552J/atm-location.png" width="400" height="250">

### 24/7s

If you select the seventh element from the list, you will be shown a list of all server 24/7s.

<img src="https://i.postimg.cc/fWq8k0Hh/247-find.png" width="300" height="250">

### Gas stations

If you select the eighth element from the list, you will be shown a list of all server gas stations.

<img src="https://i.postimg.cc/htL88J0v/gas-station.png" width="300" height="250">

### Clothing stores

If you select the ninth element from the list, you will be shown a list of all clothing stores available on the server.

<img src="https://i.postimg.cc/ZRGFF6X9/clothing-Store-find.png" width="300" height="250">

### Food stores

If you select the tenth element from the list, you will be shown a list of all food stores available on the server.
