# Other Commands

On this page you will find commands that once typed in the game, also display a new set of commands.

## /cellphonehelp

This command displays the following set of commands:

### /p(ickup)

The effect of this is to answer the phone if someone calls you. Both variants can be used: /p or /pickup.

### /h(angup)

The effect of this is to close a phone conversation. Both variants can be used: /h or /hangup.

### /call

It is used to call a player.  
To use this command you need the ID or name of the player you want to call.

_Example: /call Madalin_

### /turn

This command closes, respectively opens your phone, depending on the specified parameter:
- **/turn on** - will open the phone.
- **/turn off** - will close the phone.

### /speaker

When you are in a phone conversation, no one else around can see the conversation from the interlocutor's side, but after using the /speaker command, the messages will become visible.

### /sms

It is a command used to send text messages to other players on the server, to the specified ID or name.

_Example: /sms Madalin B-Zone!_

## /renthelp

This command displays the following set of commands:

### /enter

The /enter command is usually used to enter buildings, in this case, houses. The command is executed successfully (you will be able to enter the house) only if you are close enough to the door of that building.  
For a faster execution, you can use the equivalent of this command, namely the **"F"** key on your keyboard.

### /exit

The /exit command is usually used to exit a building, in this case, a house. The command is executed successfully (you will be able to exit the house) only if you are close enough to the exit door. For a faster execution, you can use the equivalent of this command, namely the **"F"** key on your keyboard.

### /tenants

The command will display both the number and names of all tenants online (on the server) at the time of typing the command.  
Only tenants and the house owner have access to use this command.

### /unrentroom

Using this command, the rental contract for the current home will be terminated.

## /rentcarhelp

This command displays the following set of commands:

### /unrentvehicle

This command is used to give up the rented vehicle and its keys. After using this command, you will no longer have a rented vehicle.

### /locaterentedcar

If you have a rented vehicle and forgot where you left it, then the /locaterentedcar command is useful to you. Using this command, the server will set a red point on the map representing the destination of the rented vehicle.

### /vehicles

This command displays all the vehicles you own, including rented ones so you can interact with them.

### /givekey

This command gives the keys to your vehicle to another player so they can lock or unlock it.

### /changelock

This command offers the vehicle owner the possibility to change the vehicle lock so it can no longer be used by the players to whom they gave the keys.

### /park

This command offers the vehicle owner the possibility to park the vehicle in their desired location.
- To be able to park the vehicle, it must not be hit or damaged.

## /eventhelp

This command displays the commands you can use at an event, and if you are the organizer of an event and use the command, depending on what type of event you organize, you will receive a list of commands you can use. This command displays the following set of commands when you are not the organizer/assistant of an event:

### /event

If there is an ongoing event at the time of typing the command, you will receive the following information:
- The title of the ongoing event.
- The event prize.
- The type and location of its deployment.
- The name of the organizer and their assistants.
- The level required to participate in the event.
- The minimum and maximum number of participants.

### /requestevent

This command represents a request form that level 2 helpers and B-Zone RPG server administrators can read, who will accept or not your request to organize an event. For more information about this command, access [this](/other-systems/events) page.

### /join

You join an ongoing event if it benefits from this option.

### /leaveevent

If you are a participant in an ongoing event, using this command you will leave the event.

## /leaderhelp

This command displays the following set of commands:

### /members

Using this command, you open an interactive menu with the faction members list and two separate options.  
Within the members, you can see their names, the last time they logged in, rank and FWarns.  
Regarding the 2 options, the first option is **Faction Applications**, and using this option you can open the faction applications directly in the game.  
The second option is **Faction Tests**, and using this option you can edit one of the 2 test logs of the faction.

### /factiontest

Using this command, any rank 4+ with the Tester function in the faction can start a test with a faction candidate.

### /tog

Using this command, you can find the option _Members can use faction chat_, by disabling it, no faction member will be able to use the /f command, only the leader and admins 4+.

### /fvr

By executing the command, all unoccupied faction vehicles will be respawned. The command is accessible from rank 5.

### /lc

This command is used by faction leaders to talk on the chat designated for leaders.

### /fwithdraw

By typing the /fwithdraw command without specifying any parameter, you will get information about the amount of money held at that moment by the faction safe. To withdraw money from the faction safe, the amount of money is used as a parameter.

_Example: /fwithdraw 15000_

### /fmotd

By typing the /fmotd command without specifying any parameter, the server will send you a preset message by the leader or sub-leader, if it exists. To change the preset message (message of the day), the desired message is used as a parameter.

_Example: /fmotd Everyone to HQ!_

### /fget & /fput

The command is used to put or take drugs or materials from HQ.

_Example: /fgetdrugs 100 and /fgetmaterials 100000_

### /setgc

Using this command, a mafia leader can set the minimum rank needed to send a message on /gc, the chat during alliance war.

### /vehrank

The command sets a minimum rank to use the respective faction's vehicle.

_Note: You need at least rank 6 to use this command._

## /clanhelp

This command displays the following set of commands:

### /claninvite

The /claninvite command is used by clan owners and members with rank 6+ to invite someone to their own clan.

_Note: Members with rank 6+ can only use the command on players who have an accepted application._

### /c

The command transmits a message on the [clan](https://www.rpg.b-zone.ro/shop) chat you are part of (only if you are part of one).

_Example: "/c message"_

### /clanmembers

The command opens a dialog with all clan members you are part of (only if you are part of one).

### /clanresign

With this command you can leave the clan you are in.

### /setclanowner

With this command, clan leaders can set another clan leader.

### /spray

The command is used near a clan turf to draw it in the name of your clan.

### /clanxp

The command displays a textdraw with the XP the clan has in total and how much XP it has left until it advances to the next level.

### /clanzones

The command opens a dialog with all clan turfs.

### /clanturfs

The command displays all clan turfs.

### /clancolor

This command can only be used by clan leaders to set the clan color.

### /clangoldwithdraw

With this command, clan leaders can extract Gold from the clan safe.

### /cmotd

The command is used by clan leaders to set a clan motto.

### /clanrename

With this command, clan leaders can change the names of ranks.

### /clanduty

This command will put you on duty at the clan.

### /clanleaders

The command displays all clan leaders who are on the server at that moment.

### /ctalkpower

This command prohibits using the /c command for a certain rank. (minimum rank 5+)

### /topclan

<img src="https://i.postimg.cc/fRB6X9pX/clan-top.png" width="300px">

The command displays the top clans holding turfs.

### /clanwithdraw

With this command, any member with rank 6+ can take money from the clan safe.

### /clandeposit

With this command you can deposit money in the clan safe.

### /clanput & /clanget

With this command you can put or retrieve materials or drugs.

_Example: /clanputmaterials 1000000_

### /cwithdrawpower

The command is only available to clan leaders, they can set a rank that can use the /clanwithdraw and /clanget(drugs/materials) command.

### /claimhq

The command is only available to clan leaders, with this command they can claim an available Clan HQ.

### /leavehq

With this command, clan leaders can abandon a Clan HQ.

### /interiorhq

The command is used by clan leaders to set an interior for the Clan HQ.

### /claninfo

Displays certain information related to the clan. _(only for leaders)_

### /cvs & /cvr

With /cvs clan members can spawn the clan's vehicles, with the /cvr command, a member with rank 5+ can respawn all clan vehicles.

### /buyclanvehicle

The command is used by clan leaders to buy vehicles for the clan.

### /clanwar

The command is used by members with rank 5+ or members who have been given access to it to add/remove players from the clan war.

### /guns

With this command you can choose your weapon set during clan wars.

### /leavewar

With this command you can exit your clan's war.
