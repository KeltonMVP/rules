## General Characteristics

- The Bus Driver job is a legal one.
- To become a bus driver, you need at least level 1.
- For this job, you will use the Bus vehicle.
- The job locations can be found on the map in images #1 and #2 (Los Santos and San Fierro).
- You will lose the mission if you leave the vehicle and don't return to it within 20 seconds.

## Earnings

- For each station, you will receive a random amount of money between $12 and $16.

- In-game money price: $30,000,000
- DealerShip return: $0
- Euro price at Kelton: 30 Euro
- Maximum speed: 130 KM/h
- Based on: MCI MC-8
- Number of seats: 8
- Required skill: Any skill
- Tunable: TransFender

## Specific Commands

### /work

Using this command, the server will provide you with the appropriate vehicle to start working. If you are not close enough to the designated job start location, the server will guide you to these zones by setting a red checkpoint on the map.

### /fare

Using the /fare command, you can set a specific fare for players who board the bus. The fare is an amount between $1 and $5.
